# plyder: Download Manager with Web-Interface

![Example ANSI](plyder_resources/web-interface.png)

Creating a download-manager with a focus on intuitive usage and robust architecture proved to be a fun challenge.

It ended up connecting the creation of an easy-to-use API using [FastAPI](https://fastapi.tiangolo.com/), the effective display of transmitted data using [Jinja](https://jinja.palletsprojects.com/en/2.11.x/) and an appealing rendering of the current state using [Bootstrap](https://getbootstrap.com/).
