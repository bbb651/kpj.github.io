Understanding linear regressions¶

Generate data¶

We generate synthetic data using a form of structural equation modeling. This way, we can check whether we are able to recover the coefficients.

[3]:

N = 1000

beta_g1 = 1.4
beta_g2 = -.8

mean_g1 = -2
mean_g2 = 10

[4]:

np.random.seed(42)

A = np.random.normal(size=N * 2)
B = np.r_[
    beta_g1 * A[:int(len(A)/2)] + np.random.normal(mean_g1, size=N),
    beta_g2 * A[int(len(A)/2):] + np.random.normal(mean_g2, size=N)
]
group = [1] * N + [2] * N

[5]:

df = pd.DataFrame({
    'A': A,
    'B': B,
    'group': group
})

df['group'] = df['group'].astype('category')

Fit model¶

The model:

\[Y = \beta_0 + \beta_1 \cdot group + \beta_2 \cdot A + \beta_3 \cdot A \cdot group + \epsilon\]
[6]:

mod = sm.formula.ols(formula='B ~ A * group', data=df)
fit = mod.fit()

def annotate_plot(space, mean, beta, color):
    ax.plot(space, mean + beta * space, color=color)
    ax.axhline(mean, ls='dashed', color=color)

plt.figure(figsize=(8, 6))
ax = sns.scatterplot(x='A', y='B', hue='group', data=df, s=10)

sub = df.loc[df['group'] == 1, 'A']
annotate_plot(np.linspace(sub.min(), sub.max()), fitted_mean_g1, fitted_beta_g1, sns.color_palette()[0])

sub = df.loc[df['group'] == 2, 'A']
annotate_plot(np.linspace(sub.min(), sub.max()), fitted_mean_g2, fitted_beta_g2, sns.color_palette()[1])
